Watchdog:
Класс NewFileHandler отслеживает создание новых текстовых файлов в заданной папке. Когда появляется новый файл, его содержимое передаётся в OpenAI для обработки, а ответ стримится по WebSocket обратно к клиенту.
OpenAI API:
Используется асинхронный метод acreate() для общения с OpenAI GPT-3.5. Стриминг ответа обрабатывается циклом, который отправляет данные по мере их поступления.
FastAPI и WebSocket:
Веб-сокеты позволяют в реальном времени отправлять данные обратно в браузер или другую клиентскую среду.
